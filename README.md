# Idle-battler
Idle Battler is a dynamic idle RPG with deep progression systems, weapon forging, gear crafting, and engaging combat. Build your arsenal, climb Forge Levels, and unlock cosmetics. VirusTotal verified safe (0/65 detections). Runs on budget hardware (Ryzen 3 + 6GB RAM). Free to play with optional Twitch supporter rewards. Made with Python &amp; Pygame.
