# Idle Battler

Idle Battler is a dynamic idle RPG featuring deep progression systems, weapon forging, gear crafting, and engaging combat.

Build your arsenal, climb Forge Levels, battle powerful enemies, and unlock new systems as you progress through the game.

---

## Features

- ⚔️ Weapon forging system with multiple rarity tiers
- 📈 Forge Level progression
- 🗡️ Procedurally generated weapons
- 🎯 Battle system with scaling enemies
- 🎟️ Dungeons and endgame content
- 🔁 Rebirth system for long-term progression
- 🎮 Offline rewards
- 📊 Quests and Battle Pass progression
- 🏆 Tournament system

---

## System Requirements

Idle Battler is designed to run on low-end hardware.

Minimum tested hardware:

- CPU: Ryzen 3 or equivalent
- RAM: 6GB
- Python 3.12
- OS: Windows

---
